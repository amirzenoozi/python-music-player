# Music Player
Listen To The Music With Simple Tkinter-Python Application


## Package Requirement
+ Mutagen: Read Music File Meta Tags
+ Pillow: Work With Pictures
+ PyGame: Controll Music


## Installing Package

+ ```pip3 install mutagen```
+ ```pip3 install pillow```
+ ```pip3 install pygame```
